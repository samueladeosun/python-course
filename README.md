# Python for Data Analysis - Course Repository

Welcome to the **Python for Data Analysis** course repository! This is a comprehensive introductory course on Python programming with a focus on data analysis, designed for students at NiSiF (Nigerian Students in Freiburg).

## 📚 Course Overview

This course will guide you through:
- **Python Basics** - Variables, data types, and control flow
- **Data Structures** - Lists, dictionaries, tuples, and sets
- **Functions & Modules** - Writing reusable code
- **Data Analysis with Pandas** - Working with datasets
- **Data Visualization** - Creating meaningful charts and plots
- **NumPy Essentials** - Numerical computing
- **Real-world Projects** - Applying your skills

## 🚀 Getting Started

### Prerequisites
- A computer with Python 3.8+ installed
- A code editor (VS Code, PyCharm, Jupyter Notebook, or similar)
- Basic comfort with command-line/terminal

### How to Use This Repository

1. **Read the lessons** in the `modules/` folder (Markdown files)
2. **Follow along with notebooks** in the `notebooks/` folder (Jupyter notebooks)
3. **Complete assignments** in the `assignments/` folder
4. **Check resources** for cheat sheets and helpful links

## 💾 Setup Instructions

### Installing Required Libraries

Open your terminal/command prompt and run:

```bash
pip install jupyter pandas numpy matplotlib seaborn
```

### Running Jupyter Notebooks

Navigate to the course directory and start Jupyter:

```bash
jupyter notebook
```

Then open any `.ipynb` file to start learning!
