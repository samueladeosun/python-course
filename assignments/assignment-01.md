# Assignment 1: Personal Info Display

**Objective:** Practice using variables and print statements.

## Task:
Create a Python program that:
1. Stores your personal information (name, age, city, favorite color)
2. Prints them in a formatted sentence

## Expected Output:
```
My name is [name], I am [age] years old, I live in [city], and my favorite color is [color].
```

## Solution Example:
```python
name = "Alice"
age = 25
city = "Freiburg"
favorite_color = "blue"

print(f"My name is {name}, I am {age} years old, I live in {city}, and my favorite color is {favorite_color}.")
```

## Hints:
- Use variables to store your information
- Use f-strings (f"...") for formatting
- Use print() to display the result
